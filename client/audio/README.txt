Add solemn choir music for the title screen here.

Place one of these files in this folder:
  - choir.mp3   (recommended)
  - choir.ogg   (optional, for broader browser support)

The game will try to play the music when the title screen is shown.
Many browsers require a user gesture before playing audio; if the music
does not start automatically, click or press a key on the title screen
to start it (the game will not start until you click/press again after
the first interaction in some setups).

You can find royalty-free choir or ambient music at:
  - https://pixabay.com/music/
  - https://freemusicarchive.org/
  - https://incompetech.com/

Search for "choir", "chant", or "medieval ambient".
